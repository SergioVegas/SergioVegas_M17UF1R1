# SergioVegas_M17UF1R1

## Inputs

Sutilitcen tant les tecles wasd com les fleches per moures, per canviar de gravetat es por fer amb la flecha cap a dalt o amb la w.

## Descripció joc

L'ambientació el joc és futurista, galàctica i sci-fi. Bàsicament, tracta d'un astronauta que ha fugit d'uns perseguidors i ha de trobar al seu company.
